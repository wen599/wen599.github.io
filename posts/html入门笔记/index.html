Html入门笔记

April 18, 2022

HTML入门笔记1

HTML的发明者

  • Tim Berners-Lee和同事 Daniel W. Connolly于1990年创立

HTMl起手式

        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta http-equiv="X-UA-Compatible" content="IE=edge">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Document</title>
        </head>
        <body>
            
        </body>
        </html>

  •  <!DOCTYPE html>
  • 告诉浏览器这个文件是html
  • <html lang="en">
  • en可以换成zh-CN
  • <meta charset="UTF-8">
  • 字符集是utf-8
  • <meta http-equiv="X-UA-Compatible" content="IE=edge">
  • 告诉ie使用最新的内核
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">
  • 禁用缩放，兼容手机
  • <title>Document</title>
  • 标题标签

常用表章节标签

  • h1~h6 标题标签，h1最大，h6最小
  • section 章节标签
  • p 段落标签
  • header 头部标签
  • footer 脚部标签
  • main 主要内容
  • aside 旁支内容
  • div 划分

全局属性

  • class 类属性
  • contenteditable 任何一个元素都可以被编辑
  • hidden 隐藏
  • id
  • style 样式
  • tabindex 控制tab键的切换顺序，0是最后一个，-1不到
  • title 鼠标悬停显示的内容

常用的内容标签

  • ol+li 有序列表
  • ul+li 无序列表
  • dl+dt+dd 描述列表
  • pre 保留格式
  • code 等宽字体
  • hr 分割水平线
  • br 换行
  • a 超链接
  • em 强调
  • strong 强壮
  • quote 引用
  • blockquote 换行引用
  • iframe 内嵌窗口
